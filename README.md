# food.schi.li

## Setup

Have PostgreSQL installed and running, then run

```shell
bin/setup
```

## Test

Minitest, run

```shell
bundle exec rake test
```

## Deploy

### Production

Deploy to Heroku

```shell
git push heroku master
```

Available at https://food-schi-li.herokuapp.com (or via https://food.schi.li)
